Módulo 1: Introdução ao Python e Programação

O que é Programação?

Programação é o ato de criar instruções que um computador pode executar para realizar tarefas específicas. Em redes, a programação é usada para automatizar configurações de dispositivos (como roteadores e switches Cisco), monitorar tráfego, analisar logs e gerenciar infraestruturas de rede de forma eficiente.

Python é uma linguagem de programação de alto nível, conhecida por sua sintaxe clara e legibilidade. É amplamente adotada em automação de redes devido a bibliotecas como Netmiko, Paramiko e Napalm, que facilitam a interação com dispositivos de rede. Python é interpretado, o que significa que o código é executado linha por linha, ideal para scripts rápidos em ambientes de rede.

Por que Python em Redes? Python permite reduzir tarefas manuais, como configurar VLANs em múltiplos switches, coletar dados de interfaces ou verificar o status de dispositivos, economizando tempo e minimizando erros humanos.

Instalando o Python

No Windows

  1. Baixe o instalador do Python em python.org (recomenda-se a versão mais recente, ex.: 3.11 ou superior).
  2. Execute o instalador. Marque a opção "Add Python to PATH" para facilitar o uso no Prompt de Comando.
  3. Siga as instruções do instalador, escolhendo a instalação padrão.
  4. Verifique a instalação abrindo o Prompt de Comando e digitando: python --version. Você deve ver a versão instalada (ex.: Python 3.11.7).
  5. Instale um editor de código, como Visual Studio Code, para escrever scripts Python.

No Linux

  1. Abra o terminal.
  2. Verifique se o Python 3 está instalado: python3 --version.
  3. Se não estiver, instale com: sudo apt update && sudo apt install python3 python3-pip -y (para Ubuntu/Debian). Para outras distribuições, use o gerenciador de pacotes correspondente (ex.: yum para CentOS).
  4. Verifique a instalação do pip (gerenciador de pacotes Python): pip3 --version.
  5. Considere instalar um editor como VS Code ou usar editores de terminal como Vim ou Nano.

Criando e Executando um Arquivo .py

Arquivos Python têm a extensão .py e contêm código que pode ser executado pelo interpretador Python.

  1. Crie um arquivo chamado hello.py usando um editor de texto (Notepad++, VS Code, etc.).
  2. Adicione o seguinte código: 
    print("Olá, Redes!")
  3. Salve o arquivo em uma pasta acessível.
  4. Abra o terminal/Prompt de Comando, navegue até a pasta com cd caminho/para/pasta, e execute: python hello.py (Windows) ou python3 hello.py (Linux).
  5. O resultado será: Olá, Redes!

Exemplo de Rede

Este script imprime uma mensagem de boas-vindas para administradores de rede:

print("Bem-vindo ao Python para Automação de Redes Cisco!")
print("Este script pode ser usado para gerenciar roteadores e switches.")

Módulo 2: Variáveis e Tipos de Dados

Variáveis são como contêineres que armazenam dados na memória do computador. Em Python, você não precisa declarar o tipo de variável explicitamente; o tipo é inferido automaticamente.

Tipos de Dados Comuns:

  • str: Cadeia de caracteres (texto), ex.: "Router1" ou "192.168.1.1".
  • int: Número inteiro, ex.: 80 (porta) ou 100 (VLAN).
  • float: Número decimal, ex.: 1.5 (largura de banda em Gbps).
  • bool: Valores lógicos True ou False, ex.: status de uma interface.

Em redes, variáveis são usadas para armazenar informações como endereços IP, números de portas, configurações de dispositivos e status de conexões.

Exemplos de Rede

Exemplo 1: Configuração de Interface

ip_address = "192.168.1.1"  # str
port = 80                   # int
bandwidth = 1.5             # float
is_active = True            # bool
print(f"Interface configurada com IP {ip_address}, porta {port}, banda {bandwidth} Gbps, ativa: {is_active}")

Exemplo 2: Informações de Dispositivo

device_name = "Switch1"
vlan_id = 10
uptime = 99.9
is_managed = True
print(f"Dispositivo: {device_name}, VLAN: {vlan_id}, Uptime: {uptime}%, Gerenciado: {is_managed}")

  3. Crie variáveis para um switch com nome "SW1", IP "10.0.0.2" e número de portas 24. Imprima todas.

  2. mac_address = "00:1A:2B:3C:4D:5E"
print(f"Endereço MAC do dispositivo: {mac_address}")

Módulo 3: Estruturas Condicionais

Estruturas condicionais (if, elif, else) permitem que o programa tome decisões com base em condições. Em redes, isso é útil para verificar estados de dispositivos, validar configurações ou alertar sobre falhas.

Operadores de Comparação:

  • ==: Igual a
  • !=: Diferente de
  • <, >, <=, >=: Menor, maior, menor ou igual, maior ou igual

Operadores Lógicos: and, or, not combinam condições.

Exemplos de Rede

Exemplo 1: Verificar Status de Interface

status = "up"
if status == "up":
    print("Interface está ativa!")
else:
    print("Interface está inativa.")

Exemplo 2: Validar Porta de Rede

port = 80
if port == 80:
    print("Porta HTTP detectada")
elif port == 443:
    print("Porta HTTPS detectada")
else:
    print("Porta desconhecida")

Exercícios

  1. Crie um programa que verifica se uma porta (ex.: 80) é HTTP. Se for 80, imprima "Porta HTTP". Caso contrário, imprima "Porta desconhecida".
  2. Crie um programa que verifica se a VLAN (ex.: 10) está no intervalo 1-100. Se sim, imprima "VLAN válida".
  3. Crie um programa que verifica o status de um link ("up" ou "down") e a largura de banda (ex.: 1.0). Se o link estiver "up" e a banda for maior que 0.5, imprima "Link de alta qualidade".

Soluções dos Exercícios

  1. port = 80
if port == 80:
    print("Porta HTTP")
else:
    print("Porta desconhecida")
                
  2. vlan = 10
if 1 <= vlan <= 100:
    print("VLAN válida")
else:
    print("VLAN inválida")
                
  3. status = "up"
bandwidth = 1.0
if status == "up" and bandwidth > 0.5:
    print("Link de alta qualidade")
else:
    print("Link não qualificado")

Avaliação

  1. O que faz o operador ==? (Resposta: Compara igualdade.)
  2. Escreva um if que verifica se uma interface está "down". (Resposta: if status == "down": print("Interface inativa"))
  3. Qual é a função do elif? (Resposta: Testa uma nova condição se a anterior for falsa.)

Módulo 4: Laços de Repetição

Laços de repetição permitem executar um bloco de código várias vezes. Em redes, laços são úteis para iterar sobre interfaces, VLANs ou dispositivos em uma topologia.

Tipos de Laços:

  • for: Itera sobre uma sequência (ex.: lista, intervalo).
  • while: Repete enquanto uma condição é verdadeira.

Exemplos de Rede

Exemplo 1: Verificar Interfaces

for i in range(1, 5):
    print(f"Verificando interface eth0/{i}")

Exemplo 2: Tentativas de Conexão

attempt = 1
while attempt <= 3:
    print(f"Tentando conectar ao dispositivo... Tentativa {attempt}")
    attempt += 1

Exercícios

  1. Crie um laço for que imprime "Configurando VLAN X" para VLANs de 10 a 15.
  2. Crie um laço while que tenta conectar a um dispositivo até 3 tentativas, imprimindo "Tentativa X".
  3. Crie um laço for que imprime "Interface GigabitEthernet0/X está ativa" para interfaces de 1 a 4.

Soluções dos Exercícios

  1. for vlan in range(10, 16):
    print(f"Configurando VLAN {vlan}")
                
  2. attempt = 1
while attempt <= 3:
    print(f"Tentativa {attempt}")
    attempt += 1
                
  3. for i in range(1, 5):
    print(f"Interface GigabitEthernet0/{i} está ativa")

Avaliação

  1. Qual é a diferença entre for e while? (Resposta: for itera sobre uma sequência; while repete enquanto uma condição é verdadeira.)
  2. Escreva um laço que imprime portas de 1 a 4. (Resposta: for port in range(1, 5): print(f"Porta {port}"))
  3. O que faz range(1, 5)? (Resposta: Gera uma sequência de 1 a 4.)

Módulo 5: Funções

Funções são blocos de código reutilizáveis que realizam uma tarefa específica. Elas aceitam parâmetros (entradas) e podem retornar valores. Em redes, funções são usadas para modularizar tarefas como validar IPs, verificar status de interfaces ou enviar comandos a dispositivos.

Componentes de uma Função:

  • def: Define a função.
  • Nome da função: Descritivo, ex.: check_interface.
  • Parâmetros: Entradas opcionais.
  • return: Retorna um valor (opcional).

Exemplos de Rede

Exemplo 1: Verificar Interface

def check_interface(status):
    return "Ativa" if status == "up" else "Inativa"

print(check_interface("up"))
print(check_interface("down"))

Exemplo 2: Validar Endereço IP

def is_private_ip(ip):
    return "IP privado" if ip.startswith("192.168") else "IP público"

print(is_private_ip("192.168.1.1"))
print(is_private_ip("8.8.8.8"))

Exercícios

  1. Crie uma função que recebe um número de porta e retorna "SSH" se for 22, senão "Outra porta".
  2. Crie uma função que recebe um IP e retorna "Válido" se começa com "192.168".
  3. Crie uma função que recebe o nome de uma VLAN (ex.: "VLAN10") e retorna "VLAN padrão" se for "VLAN1", senão "VLAN personalizada".

Soluções dos Exercícios

  1. def check_port(port):
    return "SSH" if port == 22 else "Outra porta"

print(check_port(22))
                
  2. def check_ip(ip):
    return "Válido" if ip.startswith("192.168") else "Inválido"

print(check_ip("192.168.1.1"))
                
  3. def check_vlan(vlan_name):
    return "VLAN padrão" if vlan_name == "VLAN1" else "VLAN personalizada"

print(check_vlan("VLAN10"))

Avaliação

  1. O que é uma função? (Resposta: Bloco de código reutilizável que executa uma tarefa.)
  2. Escreva uma função que retorna "VLAN OK" se o número da VLAN é 1. (Resposta: def check_vlan(vlan): return "VLAN OK" if vlan == 1 else "VLAN diferente")
  3. Qual é a utilidade do return? (Resposta: Envia um valor de volta para quem chamou a função.)

Módulo 6: Listas e Dicionários

Listas e dicionários são estruturas de dados que armazenam múltiplos valores. Em redes, listas podem armazenar interfaces ou VLANs, enquanto dicionários são ideais para configurações de dispositivos com pares chave-valor.

Listas: Sequências ordenadas, acessadas por índices (começando em 0).

Dicionários: Coleções de pares chave-valor, acessadas por chaves.

Exemplos de Rede

Exemplo 1: Lista de Interfaces

interfaces = ["eth0/1", "eth0/2", "eth0/3"]
print(f"Primeira interface: {interfaces[0]}")
interfaces.append("eth0/4")  # Adiciona nova interface
print(f"Interfaces atualizadas: {interfaces}")

Exemplo 2: Dicionário de Dispositivo

router = {"ip": "192.168.1.1", "model": "Cisco 2901", "ports": 4}
print(f"IP do roteador: {router['ip']}")
router["status"] = "active"  # Adiciona nova chave
print(f"Dicionário atualizado: {router}")

Exercícios

  1. Crie uma lista com 3 VLANs (10, 20, 30) e imprima a segunda VLAN.
  2. Crie um dicionário com o nome do dispositivo ("Switch1") e IP ("10.0.0.2"). Imprima o IP.
  3. Crie uma lista com 4 interfaces ("Gig0/1", "Gig0/2", "Gig0/3", "Gig0/4") e adicione uma nova interface "Gig0/5". Imprima a lista atualizada.

Soluções dos Exercícios

  1. vlans = [10, 20, 30]
print(vlans[1])
                
  2. device = {"name": "Switch1", "ip": "10.0.0.2"}
print(device["ip"])
                
  3. interfaces = ["Gig0/1", "Gig0/2", "Gig0/3", "Gig0/4"]
interfaces.append("Gig0/5")
print(interfaces)

Avaliação

  1. Qual é a diferença entre lista e dicionário? (Resposta: Lista é uma sequência ordenada; dicionário usa chaves para acessar valores.)
  2. Como acessar o segundo item de ports = ["port1", "port2", "port3"]? (Resposta: ports[1])
  3. Como adicionar uma nova chave a um dicionário? (Resposta: dict["nova_chave"] = valor)

Módulo 7: Manipulação de Arquivos

Python permite ler e escrever arquivos, o que é essencial em redes para gerenciar configurações, salvar logs ou exportar dados de dispositivos. Arquivos podem ser abertos em modos como "r" (leitura), "w" (escrita, sobrescreve) ou "a" (escrita, adiciona).

Usar a instrução with garante que o arquivo seja fechado automaticamente após o uso.

Exemplos de Rede

Exemplo 1: Criar Configuração

with open("config.txt", "w") as file:
    file.write("interface eth0/1\nip address 192.168.1.1 255.255.255.0")

with open("config.txt", "r") as file:
    print(file.read())

Exemplo 2: Registrar IPs

ips = ["192.168.1.1", "10.0.0.2", "172.16.0.1"]
with open("ip_list.txt", "w") as file:
    for ip in ips:
        file.write(f"{ip}\n")

with open("ip_list.txt", "r") as file:
    for line in file:
        print(f"IP lido: {line.strip()}")

Exercícios

  1. Crie um arquivo vlans.txt com a lista de VLANs "VLAN 10, VLAN 20".
  2. Leia e imprima o conteúdo de vlans.txt.
  3. Crie um arquivo interfaces.txt com duas interfaces ("Gig0/1", "Gig0/2") e adicione uma terceira ("Gig0/3") usando o modo "a".

Soluções dos Exercícios

  1. with open("vlans.txt", "w") as file:
    file.write("VLAN 10, VLAN 20")
                
  2. with open("vlans.txt", "r") as file:
    print(file.read())
                
  3. with open("interfaces.txt", "w") as file:
    file.write("Gig0/1\nGig0/2\n")
with open("interfaces.txt", "a") as file:
    file.write("Gig0/3\n")

Avaliação

  1. O que faz o modo "w" ao abrir um arquivo? (Resposta: Escreve, sobrescrevendo o arquivo.)
  2. Como ler um arquivo chamado log.txt? (Resposta: with open("log.txt", "r") as file: print(file.read()))
  3. Qual é a diferença entre os modos "w" e "a"? (Resposta: "w" sobrescreve; "a" adiciona ao final.)

Módulo 8: Introdução à Automação de Redes com Python

Automatizar tarefas de rede com Python reduz o trabalho manual e aumenta a eficiência. Bibliotecas como telnetlib (para Telnet) e paramiko (para SSH) permitem interagir com dispositivos Cisco, enviando comandos e coletando outputs.

Nota: Use ambientes de teste (ex.: Cisco Packet Tracer) para evitar impactos em redes de produção.

Exemplos de Rede

Exemplo 1: Usando telnetlib

import telnetlib

host = "192.168.1.1"
user = "admin"
password = "cisco"

tn = telnetlib.Telnet(host)
tn.read_until(b"Username: ")
tn.write(user.encode("ascii") + b"\n")
tn.read_until(b"Password: ")
tn.write(password.encode("ascii") + b"\n")
tn.write(b"show version\n")
tn.write(b"exit\n")
print(tn.read_all().decode("ascii"))

Exemplo 2: Verificar Interfaces

import telnetlib

host = "192.168.1.1"
user = "admin"
password = "cisco"

tn = telnetlib.Telnet(host)
tn.read_until(b"Username: ")

tn.read_until(b"Password: ")
tn.write(password.encode("ascii") + b"\n")
tn.write(b"show ip interface brief\n")
tn.write(b"exit\n")
output = tn.read_all().decode("ascii")
print("Status das interfaces:")
print(output)

Módulo 9: Automação Avançada com Netmiko

Netmiko é uma biblioteca Python que simplifica a automação de redes via SSH, suportando dispositivos Cisco e outros. É mais robusta que telnetlib e ideal para configurar dispositivos, coletar dados e gerenciar redes.

Instalação: pip install netmiko

Funcionalidades:

  • Enviar comandos de show (send_command).
  • Enviar configurações (send_config_set).
  • Gerenciar conexões SSH com dispositivos de rede.

Exemplos de Rede

Exemplo 1: Coletar Informações

from netmiko import ConnectHandler

device = {
    "device_type": "cisco_ios",
    "host": "192.168.1.1",
    "username": "admin",
    "password": "cisco",
}

connection = ConnectHandler(**device)
output = connection.send_command("show version")
print(output)
connection.disconnect()

Exemplo 2: Configurar Interface

from netmiko import ConnectHandler

device = {
    "device_type": "cisco_ios",
    "host": "192.168.1.1",
    "username": "admin",
    "password": "cisco",
}

connection = ConnectHandler(**device)
commands = [
    "interface GigabitEthernet0/1",
    "ip address 192.168.2.1 255.255.255.0",
    "no shutdown"
]
connection.send_config_set(commands)
connection.send_command("write memory")
print("Interface configurada!")
connection.disconnect()

device = {
