Módulo 1: Introdução ao Python e Programação

Objetivos

  • Entender o que é Python e sua aplicação em redes.
  • Configurar o ambiente de desenvolvimento no Windows e Linux.
  • Criar, salvar e executar o primeiro programa em Python usando um editor/IDE.

Conteúdo

O que é Python?

Python é uma linguagem de programação de alto nível, interpretada, de sintaxe simples e legível. Em redes, é amplamente usada para:

  • Automação: Configurar dispositivos Cisco via scripts.
  • Análise de Logs: Processar logs de roteadores/switches.
  • Monitoramento: Verificar status de interfaces ou dispositivos.

Instalação do Python

No Windows
  1. Acesse python.org/downloads.
  2. Baixe a versão mais recente (e.g., Python 3.11 ou superior).
  3. Execute o instalador:
    • Marque a opção Add Python to PATH.
    • Escolha Install Now.
  4. Verifique a instalação:
    • Abra o Prompt de Comando (tecle cmd no menu Iniciar).
    • Digite python --version e pressione Enter. Você deve ver algo como Python 3.11.x.
  5. Instale o pip (gerenciador de pacotes):
    • Digite python -m ensurepip --upgrade.
    • Verifique com pip --version.
No Linux (Ubuntu/Debian)
  1. Abra o terminal (Ctrl + Alt + T).
  2. Atualize os pacotes: sudo apt update.
  3. Instale o Python: sudo apt install python3 python3-pip -y.
  4. Verifique a instalação:
    • Digite python3 --version. Você deve ver algo como Python 3.10.x ou superior.
    • Verifique o pip: pip3 --version.
  5. (Opcional) Instale o gerenciador de pacotes: sudo apt install python3-venv -y.

Configurando um Editor/IDE

Recomendamos o Visual Studio Code (VS Code) por sua simplicidade e suporte a Python.

Instalação do VS Code:
  • Windows: Baixe em code.visualstudio.com e execute o instalador.
  • Linux: No Ubuntu, use: sudo snap install --classic code ou baixe o .deb do site.
Configurando o VS Code para Python:
  1. Abra o VS Code.
  2. Instale a extensão Python:
    • Clique no ícone de extensões (ou Ctrl + Shift + X).
    • Pesquise por "Python" (da Microsoft) e clique em Install.
  3. Selecione o interpretador Python:
    • Pressione Ctrl + Shift + P, digite Python: Select Interpreter.
    • Escolha a versão instalada (e.g., Python 3.11).

Criando, Salvando e Executando um Script

Criando um Arquivo:
  1. No VS Code, clique em File > New File.
  2. Escreva o seguinte código:
print("Bem-vindo à automação de redes com Python!")
Salvando o Arquivo:
  1. Clique em File > Save (ou Ctrl + S).
  2. Escolha um local (e.g., C:\Users\SeuNome\PythonScripts no Windows ou ~/python_scripts no Linux).
  3. Nomeie o arquivo como hello.py (a extensão .py é obrigatória).
Executando o Script:

Via VS Code:

  • Clique no botão "Run" (triângulo verde no canto superior direito) ou pressione F5.
  • A saída aparecerá no terminal integrado do VS Code.

Via Terminal:

  1. Abra o terminal no VS Code (Ctrl + `) ou use o Prompt de Comando/Terminal do sistema.
  2. Navegue até o diretório do arquivo:
    • Windows: cd C:\Users\SeuNome\PythonScripts
    • Linux: cd ~/python_scripts
  3. Execute: python hello.py (Windows) ou python3 hello.py (Linux).
  4. A saída será: Bem-vindo à automação de redes com Python!

Exibir uma mensagem com o nome de um roteador Cisco.

print(f"Configurando o roteador {router_name}")

Módulo 2: Variáveis e Tipos de Dados

Objetivos

  • Compreender variáveis e tipos de dados.
  • Aplicar variáveis em cenários de redes Cisco.

Conteúdo

Variáveis: Espaços na memória para armazenar dados. Regras:

  • Nomes devem começar com letras ou _ (e.g., ip_address, _port).
  • Não use palavras reservadas (e.g., if, for).

Exemplo:

ip_address = "192.168.1.1"

Tipos de Dados:

  • Inteiro (int): Números inteiros, como portas ou IDs de VLAN.
    • port = 80
vlan_id = 100
  • String (str): Texto, como nomes de dispositivos ou configurações.
    • device = "Cisco IOS"
  • Booleano (bool): Verdadeiro (True) ou falso (False), útil para status.
    • is_active = True
  • Float (float): Números decimais, como largura de banda.
    • bandwidth = 1.5  # 1.5 Gbps

Operações Básicas:

  • Concatenar strings: device + " ativo"
  • Aritmética com inteiros/floats: port + 1, bandwidth * 1000

Exemplo:

vlan_id = 10
vlan_id = vlan_id + 1  # Incrementa para 11
print(f"Nova VLAN: {vlan_id}")

Exemplo de Rede

Armazenar e exibir informações de um dispositivo.

hostname = "Router1"
ip_address = "10.0.0.1"
is_online = True
ports = 24
print(f"Dispositivo: {hostname}, IP: {ip_address}, Online: {is_online}, Portas: {ports}")

Como Criar/Salvar/Executar:

  1. Crie um arquivo device_info.py no VS Code.
  2. Cole o código acima.
  3. Salve em seu diretório de scripts (e.g., C:\Users\SeuNome\PythonScripts).
  4. Execute via VS Code (botão "Run") ou terminal: python device_info.py.

Módulo 3: Estruturas Condicionais

Objetivos

  • Usar estruturas if, elif e else.
  • Aplicar condicionais em decisões de rede.

Conteúdo

Estruturas Condicionais:

Usadas para tomar decisões com base em condições.

Sintaxe:

if condição:
    # Código
elif outra_condição:
    # Código
else:
    # Código

Operadores de Comparação:

  • Igual: ==
  • Diferente: !=
  • Maior/Menor: >, <, >=, <=

Exemplo:

port = 80
if port == 80:
    print("Porta HTTP detectada")

Operadores Lógicos:

  • and, or, not

Exemplo:

status = "up"
speed = 1000
if status == "up" and speed >= 1000:
    print("Interface rápida e ativa")

Exemplo de Rede

Verificar o status de uma interface.

interface_status = "up"
if interface_status == "up":

Módulo 4: Laços de Repetição

Objetivos

  • Utilizar laços for e while.
  • Aplicar laços em tarefas repetitivas de redes.

Conteúdo

Laço for:

Itera sobre uma sequência (lista, intervalo, etc.).

Sintaxe:

for item in sequência:
    # Código

Exemplo com range:

for i in range(1, 5):  # De 1 a 4
    print(f"Porta {i}")

Laço while:

Executa enquanto uma condição for verdadeira.

Sintaxe:

while condição:
    # Código

Exemplo:

count = 1
while count <= 3:
    print(f"Verificando dispositivo {count}")
    count += 1

Exemplo de Rede

Listar interfaces de um dispositivo.

interfaces = ["Gig0/1", "Gig0/2", "Gig0/3"]
for interface in interfaces:
    print(f"Configurando {interface}")

Conteúdo

Definindo Funções:

Funções permitem reutilizar código.

Sintaxe:

def nome_funcao(parametros):
    # Código
    return resultado

Parâmetros e Retorno:

  • Parâmetros: Dados passados para a função.
  • Retorno: Resultado devolvido com return.

Exemplo:

def double_port(port):
    return port * 2
print(double_port(80))  # Saída: 160

Exemplo de Rede

Listas:

Coleções ordenadas de itens.

Métodos comuns: append(), remove(), pop().

devices = ["R1", "R2", "SW1"]

router = {"hostname": "R1", "ip": "10.0.0.1", "ports": 24}
print(router["ip"])  # Acessa 10.0.0.1

Módulo 7: Manipulação de Arquivos

Objetivos

  • Ler e escrever arquivos.
  • Aplicar manipulação de arquivos em logs de rede.

Conteúdo

Leitura de Arquivos:

Usa open() com modo "r" (read).

Exemplo:

with open("config.txt", "r") as file:
    content = file.read()
    print(content)

Escrita de Arquivos:

Usa open() com modo "w" (write) ou "a" (append).

Exemplo:

with open("config.txt", "w") as file:
    file.write("interface Gig0/1\n ip address 192.168.1.1")

Exemplo de Rede

Salvar configuração de um roteador.

config = "hostname R1\ninterface Gig0/1\n ip address 10.0.0.1 255.255.255.0"
with open("router_config.txt", "w") as file:

Módulo 8: Introdução à Automação de Redes com Python

Objetivos

  • Usar bibliotecas como netmiko para automação.
  • Conectar a dispositivos Cisco via SSH.

Conteúdo

Biblioteca Netmiko:

Simplifica conexões SSH a dispositivos de rede.

Instalação:

No terminal, execute: pip install netmiko (Windows) ou pip3 install netmiko (Linux).

Exemplo:

from netmiko import ConnectHandler

device = {
    "device_type": "cisco_ios",
    "ip": "192.168.1.1",
    "username": "admin",
    "password": "cisco"
}

connection = ConnectHandler(**device)
output = connection.send_command("show ip int brief")
print(output)
connection.disconnect()

Exemplo de Rede

Exibir e salvar configurações de um roteador Cisco.

from netmiko import ConnectHandler

device = {
    "device_type": "cisco_ios",
    "ip": "192.168.1.1",
    "username": "admin",
    "password": "cisco"
}

connection = ConnectHandler(**device)
output = connection.send_command("show running-config")
with open("running_config.txt", "w") as file:
    file.write(output)
connection.disconnect()

Como Criar/Salvar/Executar:

  1. Crie um arquivo netmiko_config.py no VS Code.
  2. Cole o código acima.
  3. Salve em seu diretório de scripts.
  4. Execute: python netmiko_config.py.
  5. Nota: Teste em um ambiente simulado (e.g., GNS3, Cisco Packet Tracer) com um dispositivo configurado.

Referências

  • Netmiko Documentation
  • Cisco DevNet: Netmiko

Avaliação

  1. Instale a biblioteca netmiko usando pip.
  2. Crie um arquivo show_version.py que use netmiko para conectar a um dispositivo simulado (use GNS3 ou Cisco Packet Tracer) e execute show version.
